# Sample BlockProvider project

This BlockProvider exposes :

* a block that displays bitcoin current value

## Setup

```
npm install
```

## Start it

```
npm start
```

## Check how it renders locally

Checkout [BlockProvider Runner](../blockprovider-runner).

## Develop

```
npm run start:dev
```
